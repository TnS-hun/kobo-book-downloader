CAPTCHA Code Instructions

  • Open https://authorize.kobo.com/signin in a private or incognito window in your browser. Do not login!
  • Open the developer tools. Press F12 in Firefox/Chrome, or right click the window and select "Inspect"
  • Select the console tab, and copy-and-pase the following code, then press enter.
  • A captcha should show up below the Sign-in form. Once you solve the captcha its response will be written below the pasted code in the browser's console. Copy the response (the line below "Captcha response:") and paste it here.
var newCaptchaDiv = document.createElement("div"); newCaptchaDiv.id = "new-hcaptcha-container"; var siteKey = document.getElementById('hcaptcha-container') .getAttribute('data-sitekey'); document.body.replaceChildren(newCaptchaDiv); grecaptcha.render( newCaptchaDiv.id, { sitekey: siteKey, callback: function(r) { console.log("Captcha response:"); console.log(r); } }); console.log('Click the checkbox to get the code');
